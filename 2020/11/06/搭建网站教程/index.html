搭建个人网站

发表于 2020-11-06 更新于 2020-11-08 分类于 网站 阅读次数：

搭建

下载Git和Node.js

  1. Git：安装完成后在Windows PowerShell中输入git --version验证是否安装成功。

  2. Node.js：安装好后在Windows PowerShell输入node -v和npm -v验证是否安装成功。

    配置Git

  3. 在windows菜单中，打开Git Bash

  4. 输入用户名和邮箱

  5. 生成新的SSH Key

    一直回车直到出现

  6. 打开github，点击头像→setting→SSH and GPG keys→New SSH key，输入Title，再在Git Bash中输入cat ~/.ssh/id_rsa.pub后，复制出现的内容到Key里，点击Add SSH key

  7. 在Git Bash中输入ssh -T git@github.com,再输入yes，出现自己的用户名，表示配置成功

安装Hexo

  1. 利用npm命令安装hexo，最后通过 hexo -v验证是否安装成功。

  2. 在Github上创建Repository，Repository name 中填入username.github.io，点击Create repository

    有很多教程中，在此处可以接着：

    在Repository的Setting中的GitHub Pages部分，Choose a theme先选择一个主题，直接可以生成Github主题的网站，再进行后续操作。

    但在2020-10-01后，github新建的默认的库位main而不是master，会导致在hexo g -d后，网页一直不更新，此时就需要在GitHub Pages部分的Source中把分支切换为master。

    为了避免这个问题，可以跳过这一步；也可以在deploy:中把branch: master改成branch: main。

  3. 在本地创建一个与 Repository 中博客项目同名的文件夹username.github.io后，进入文件夹，右键 Git bash here，输入hexo init初始化文件夹，接着输入npm install安装必备的组件

  4. 确保git部署npm install hexo-deployer-git –save

  5. 输入hexo g生成静态网页，再输入hexo s打开本地服务器，然后打开浏览器预览：http://localhost:4000/

  6. 按ctrl+c关闭本地服务器。

在文章中插入图片

  1. 如果只有少量图片，可以在~/source目录下新建images文件夹，通过![](/images/xxx.png)来引用。
  2. 如果图片比较多，可以放建立文章自己的目录，首先在站点配置文件中，将post_asset_folder关键词改成true。之后通过hexo new post “article title”在新建文章时，就会出现与文章名相同的文件夹article title，将图片放入其中后，通过![](XXXX.jpg)直接引用。

以上可以在网页中正确显示，但是在md编辑器，比如Typora中无法显示图片，此时可以在md文件的Front-matter部分，根据文章所有图片存在的位置，添加typora-root-url: ..或者typora-root-url: article title，即可在编辑器中也正常显示图片。

另外推荐第二种方法， 因为在Typora的偏好设置中，图像部分，选择复制到指定路径， ./${filename}，之后可以直接在编辑md文件的时候复制图片，文章中会显示图像，并且已经在article title的文件夹下复制了该图片，可以直接在网页上显示。

修改主题

  1. 选择主题的github项目，然后点击Cod→SSH，复制后，在~/themes文件夹下右键，GIt Bash here，然后输入：

  2. 打开站点配置文件，将theme后面修改为该主题的名字

  3. 在主题配置文件~/themes/hexo-theme-next/_config.yml中，在Schemes部分可以选择不同的风格。

修改菜单

  1. 打开主题配置文件，在menu部分设定菜单的内容，格式为Key: /link/ || icon

    • key如果如果有默认翻译，会自动加载翻译，如果不存在就不会加载。翻译可以在\themes\hexo-theme-next\languages的.yml文件中修改。
    • ||前面是菜单的链接，后面是菜单的Font Awesome图标。

  2. menu_settings中，可以通过修改icons是否显示菜单的图标

设置头像

打开主题配置文件，在avatar部分设置头像的链接。

  • url中可以是完整的互联网 URI，也可以将头像放置在source/images/ 目录下
  • 修改rounded，头像可以为圆的
  • 修改rotated，鼠标移动到头像上头像会旋转

设置网站访问量

  1. 登陆百度统计，新增网站，网站域名和网站首页都填网站的地址即可。

  2. 在 hm.src中，复制 hm.js? 后面那串统计脚本 id到主题配置文件中baidu_analytics后面

  3. 修改完参数后执行 hexo g 和 hexo d 命令部署博客。在百度统计页面，点击代码检查，当出现”页面代码安装状态：代码安装正确“时，表示配置成功。

  4. 在主题配置文件中，将enable修改为true，就可以在页面添加统计模块。

添加文章阅读量统计功能

  1. 注册并登陆LeanCloud，创建应用，输入应用名称后创建

  2. 点击储存图标→创建 Class来新建Class用来专门保存我们博客的文章访问量等数据

    • Next要求Class名称必须为Counter，所以命名一定要为Counter
    • 为了防止阅读量统计不准确，建议ACL权限设置为无限制，然后点击创建Class。

  3. 点击齿轮图标→应用Key，复制AppID，AppKey，在主题配置文件中leancloud_visitors部分修改：

  4. 由于app-id和app-key完全暴露，为了防止他人使用，在齿轮图标→安全中心中的Web 安全域名里，增加自己的网站地址并保存。

注意：记录文章访问量的唯一标识符是文章的发布日期以及文章的标题，因此请确保这两个数值组合的唯一性，如果你更改了这两个数值，会造成文章阅读数值的清零重计。

按以上步骤添加完毕后，点开文章会出现两个阅读次数，怀疑是设置网站访问量的同时已经设置了文章阅读量的统计，因此暂时删去。
